# kalshi-snapshots

Public read-only snapshots of the Kalshi paper-trading dashboard. Source
code + trade DBs live in a private repo; this one only mirrors the static
HTML snapshot for remote viewing.

## View

- **Latest:** [index.html](index.html) (always-current pointer)
- **By date:** see snapshot_*.html files

Render via any of:
- GitHub source view (works on phone): https://github.com/dcamco/kalshi-snapshots/blob/main/index.html
- htmlpreview rendered: https://htmlpreview.github.io/?https://github.com/dcamco/kalshi-snapshots/blob/main/index.html
- Raw download: https://raw.githubusercontent.com/dcamco/kalshi-snapshots/main/index.html

Snapshots are regenerated by `scripts/snapshot_dashboard.py` in the main
repo and pushed here.
